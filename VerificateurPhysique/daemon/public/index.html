Invitation NFC


Cet application génére les adresses didcomm:// d'invitaion, les raccourcit en utilisant l'app d'URL courte en format deeplink.


Pour voir le fonctionnement, merci de vous référer à la doc dans Github


L'endpoint suivant génère un deeplink selon le standard didcomm:// https://exp-port-e-invitant.apps.exp.openshift.cqen.ca/connection.
